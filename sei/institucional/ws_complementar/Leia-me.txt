Roteiro de instalação do Módulo do SEI "ws_complementar":

1 - Caso ainda não exista, criar a pasta "institucional" na raiz do SEI (é mandatório a manutenção deste nome de pasta conforme o padrão do SEI para módulos):
	
	/sei/institucional
	
2- Copiar a pasta de código "ws_complementar" para a pasta sei/institucional:

	/sei/institucional/ws_complementar 

3- Modificar o arquivo ConfiguracaoSEI.php para reconhecer o módulo:
	
	Ex. 'SEI' => array(
 	          'URL' => 'http://{nome_servidor}/sei/',
 	          'Producao' => true,
 	          'RepositorioArquivos' => '/sei-nfs/dados',
 	      	  'Modulos' => array(
 	      				'WScomplementar' =>dirname(__FILE__).'/institucional/ws_complementar',
 	      		),

4- O endereço do WSDL é o seguinte:

	http://[dominio_servidor]/sei/controlador_ws.php?servico=wscomplementar
	
5- Manual e demais orientações constam na Wiki do Projeto Principal: https://softwarepublico.gov.br/gitlab/anatel/mod-sei-wscomplementar/wikis/home

6- Orientamos que este arquivo "Leia-me.txt" não seja mantido no servidor.

7- O projeto desse módulo é de desenvolvimento colaborativo, devendo seguir a metodologia definida, especialmente abrindo Issue antes de qualquer desenvolvimento. Link para o projeto principal: https://softwarepublico.gov.br/gitlab/anatel/mod-sei-wscomplementar
